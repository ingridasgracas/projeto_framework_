ATIVIDADE 1 - Simulação de instalação/configuração do Tomcat 8.x e NetBeans com Spring MVC
---------------------------------------------------------------------------------------
PASSOS EXECUTADOS (SIMULADO):
1. Baixei o zip apache-tomcat-8.5.82 (simulado).
2. Extraí para C:/apache-tomcat-8.5.82 (simulado).
3. No NetBeans: File > New Project > Java with Ant > Web Application.
4. Configurei server: Apache Tomcat (Server Location: C:/apache-tomcat-8.5.82).
5. Marquei Spring Web MVC; criei um projeto chamado DemoSpringNetBeans.
6. Deploy (simulado) - servidor Tomcat iniciado e aplicação acessível em http://localhost:8080/DemoSpringNetBeans/.

RESULTADO (SIMULADO):
- Aplicação inicializada com Spring MVC.
- Página index.htm servida via Tomcat (conteúdo incluído em src/main/webapp/index.htm).
- Logs simulados incluídos em tomcat_simulated.log.
